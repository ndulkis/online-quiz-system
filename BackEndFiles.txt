BACKEND PHP FILES - COMPLETE
================================

All backend PHP files have been created and are ready to use.

FILES CREATED
-------------

1. auth.php
   - Handles user login and registration
   - Uses prepared statements for security
   - Returns JSON responses

2. get_quiz_questions.php
   - Fetches a specific quiz and all its questions
   - Used when user starts taking a quiz

3. get_quizzes.php
   - Returns list of available quizzes
   - Can filter by category

4. get_user_stats.php
   - Gets user's statistics (quizzes taken, average score, rank)
   - Used on the dashboard

5. get_recent_activity.php
   - Shows user's 5 most recent quiz attempts
   - Used on the dashboard

6. submit_quiz.php
   - Saves quiz results when user completes a quiz
   - Calculates score automatically
   - Saves both the attempt and individual answers

7. get_leaderboard.php
   - Returns top 10 users by average score
   - Supports global, monthly, and weekly periods

8. save_quiz.php
   - Creates new quizzes
   - Saves questions with the quiz
   - Used by the "Create Quiz" page

9. get_user_profile.php
   - Gets user profile information
   - Includes stats like total time spent, highest score
   - Used on the profile page

HOW THEY WORK
-------------
All files:
- Connect to MySQL using PDO
- Use prepared statements (secure against SQL injection)
- Return JSON responses
- Have simple comments explaining what's happening

TESTING THE SYSTEM
------------------
To test everything:

1. Start the server:
   cd "assets"
   php -S localhost:8000

2. Open in browser:
   http://localhost:8000

3. Login with:
   Email: john@example.com
   Password: password123

4. You should be able to:
   - See your stats on the dashboard
   - View available quizzes
   - Take a quiz
   - Submit and see results
   - View the leaderboard
   - See your profile

WHAT'S CONNECTED
----------------
Database -> PHP Backend -> HTML/JavaScript Frontend

The frontend (index.html) makes requests to these PHP files,
which query the database and return the data as JSON.
